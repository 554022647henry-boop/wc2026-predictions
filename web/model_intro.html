03 · 评分体系

7 个维度 · 独立评分 · 差异化权重

每个维度由一个专属 Agent 独立评分，杜绝单一模型「什么都想考虑」导致的因素叠加偏差。

04 · 量化模型

加权分差 → 概率分布

所有维度评分通过置信度加权后合并为一个标量 δ，再经过针对国际赛事校准的概率映射表转换为三方概率。

加权分差 Weighted Delta
δ  =  Σ  ( wi  ·  ci  ·  (Ai − Bi) ) Σ  ( wi  ·  ci )
wi = 维度权重  |  ci = 信息置信度（0–10） |  评分 ∈ [4, 8]，δ > 0 表示 A 队占优

δ 到概率映射——已针对国际赛事校准

📉  赔率校验：若模型概率与市场隐含概率偏差 > 20%，触发置信度警告——不改变预测方向，但会降低 confidence 标签
